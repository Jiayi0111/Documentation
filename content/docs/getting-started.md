## What you need 

### Hardware
* An ESP32 board. (with our extension board)
* USB cable - USB A / micro USB B.
* Computer running Windows, Linux, or macOS.

### Software
* Visual Studio Code （recommended） or Eclipse
* ESP-IDF plugin
